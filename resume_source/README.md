# 이력서 원본 파일 업로드 폴더

이 폴더(`resume_source/`)는 **AI 에이전트가 귀하의 이력서를 자동으로 파싱(Parsing)하여 코드로 변환**하기 위해 사용하는 공간입니다.

## 🤖 AI 에이전트 자동 업데이트 사용 방법

1. **파일 업로드**:
   본인의 기존 이력서를 이 폴더 안에 넣습니다.
   지원하는 확장자는 다음과 같습니다:
   - `.md` (가장 권장됨 - 노션에서 마크다운 내보내기 한 파일 등)
   - `.txt` (일반 텍스트)
   - `.pdf` (텍스트 추출 가능한 형태)

2. **파일 이름 규칙**:
   어떤 이름이든 상관없으나 가급적 `my_resume.md` 형태를 권장합니다.

3. **에이전트에게 명령하기**:
   채팅창에 아래와 같이 에이전트에게 지시해 주세요.
   > *"내 이력서를 `resume_source/` 폴더에 올렸어. 이거 읽고 내 정보로 싹 업데이트 해줘."*

4. **에이전트가 수행하는 작업**:
   에이전트가 이 폴더의 파일을 읽은 후, `src/data/` 내부의 다음 파일들을 당신의 정보에 맞게 자동으로 재작성합니다:
   - `profile.ts` (인적사항, 학력)
   - `experience.ts` (경력, 핵심 성과 수치 Bold 처리)
   - `projects.ts` (진행한 프로젝트 상세 정보)
   - `aiCases.ts` (AI 활용 사례)
